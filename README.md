# PsyProAi 🛡️
**The Neural Shield against Social Engineering**

PsyProAi is an active counter-intelligence platform built to bridge the human gap in cybersecurity. Traditional antivirus software protects devices, but PsyProAi protects human psychology. By intercepting live audio and clipboard data, the system detects emotional coercion, false urgency, and financial fraud in real-time, allowing users to deploy AI-generated scam-bait to neutralize attackers.

## 🧠 System Architecture (The 4 Layers)
* **Layer 1 (Edge):** Live Call Interceptor - Captures environmental audio to detect coercive speech.
* **Layer 2 (Brain):** Neural Analysis - Powered by the Google Gemini API to scan text and audio for psychological manipulation.
* **Layer 3 (Sentinel):** Active Decoy - Generates context-aware "Scam-Bait" to waste the attacker's time.
* **Layer 4 (Deception):** Physical Trap - Arms the system for lockdown against severe threats.

## 🚀 Technologies Used
* **Python 3.x**
* **Google Gemini API** (Core Neural Engine)
* **CustomTkinter** (Hardware-accelerated UI)
* **SpeechRecognition** (Live Audio Transcription)
* **Pyperclip** (Clipboard Threat Scanning)

## ⚙️ How to Run the Prototype
1. Clone this repository to your local machine.
2. Install the required dependencies: `pip install -r requirements.txt`
3. Create a `.env` file in the root directory and add your Gemini API Key: `GEMINI_API_KEY=your_api_key_here`
4. Run the main application: `python start_PsyProAi.py`
